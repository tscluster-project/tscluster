User Guide
==========

We provide tutorial notebooks and Colab notebooks with example codes to try working with tscluster:

- The Tutorial section contains notebooks with example codes and outputs for using tscluster with real world data and synthetic data (useful for working with small examples to get an overview of the modules in tscluster).
- The Colab notebooks link to a Google Colab session to try the example codes in the tutorial section.
